## 它到底能做什么

DisplayBuddy 支持 Mac 和 Windows 双平台，主要功能围绕「显示器参数控制」展开：

**亮度、对比度、音量调节**：直接从菜单栏操作外接显示器，也支持用键盘的亮度键和音量键直接控制，不用摸显示器本体。兼容 LG、戴尔、宏碁、AOC、BenQ、华硕等主流品牌，只要显示器支持 DDC/CI 协议基本都能用。

**预设管理**：白天一套配置、夜晚一套配置、看片一套配置，保存好之后一键切换。用多了会觉得这是刚需，毕竟每次手调太麻烦了。

**多显示器同步**：有两台外接显示器的用户肯定懂，两边亮度不一致看着很割裂。DisplayBuddy 可以一次同步多台屏幕的亮度和对比度，省去逐一调整的麻烦。

**输入源切换**：从任务栏就能把显示器的信号源从 HDMI 切到 DP、USB-C，不用再去显示器菜单里找。这个功能对经常在不同设备间切换显示器的人来说挺实用。

**Mac 专属：亮度同步 + UltraBright + Siri 支持**：Mac 版有几个额外的好东西——MacBook 内屏和外接显示器亮度可以联动同步；UltraBright 模式能解锁 XDR 屏幕的峰值亮度，在大太阳下办公的用户可能会用到；还支持 Siri 和快捷指令，直接喊一声或者设个自动化脚本就能调屏幕参数，有点未来感。

**macOS Sonoma 小组件**：可以直接在桌面上放一个 DisplayBuddy 小组件，在全屏工作状态下不用切换应用就能激活预设。

## 谁适合买

说实话，DisplayBuddy 不是一款「每个人都需要」的软件，但如果你对下面几条有共鸣，入手基本不会后悔：

- **接了外接显示器的 Mac 或 Windows 用户**：这是最核心的受众，Mac 用户尤其明显——macOS 原生对第三方显示器的支持很弱，连基础亮度调节都没有，DisplayBuddy 补上了这个空缺。
- **多显示器工作流**：设计师、开发者、视频剪辑师这类「双屏甚至三屏」用户，多台屏幕的参数管理会是一个持续的痛点。
- **经常在不同场景下工作的人**：在家办公、咖啡厅、会议室来回切，每次重新调显示器参数会很烦，预设功能在这里就体现了价值。

如果你只用 MacBook 不接任何外接显示器，那可能用处不大，毕竟 macOS 自带的内屏亮度调节已经够用了。
